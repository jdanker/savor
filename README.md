# Savor

A simple iOS app for tracking restaurants you want to try and places you've been.

## Features

- **Search & Save** — Find restaurants using Google Places autocomplete and add them to your list
- **Visit Tracking** — Mark places as "Want to Try" or "Been" with swipe actions
- **Restaurant Details** — View ratings, price level, cuisine type, editorial summaries, and photo carousels
- **Drag & Drop** — Reorder your list however you like

## Tech Stack

- SwiftUI + Observation framework
- Google Places SDK
- JSON file persistence
- Two-tier photo caching (memory + disk)
- iOS 26+
